# QA Portfolio: AI学習レコメンダー

中高生向けAI学習レコメンド機能のQAポートフォリオ。
PdM・開発・QAのロールを1人で担い、QAプロセスを一気通貫で設計・実装した作例。

## このリポジトリについて

PRDから始まり、テスト計画・テスト設計・自動テスト・テストレポートまでを
実際のアジャイル開発のワークフローに沿って構築している。

コミット履歴では、各コミットメッセージのプレフィックスでロールを示している:
- `[PdM]` プロダクトマネージャーとしての成果物
- `[Dev]` 開発エンジニアとしての成果物
- `[QA]` QAエンジニアとしての成果物

## 構成

- `docs/` — PRD、テスト計画書、テスト設計書、テストレポート
- `src/` — SUT（System Under Test、レコメンダーアプリ本体）
- `tests/e2e/` — Playwright E2Eテスト
- `tests/ai-quality/` — AI出力品質テスト
